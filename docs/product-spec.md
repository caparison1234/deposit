### 2-1. `/` — 오늘의 금리 랭킹 (홈)

**목적**: 방문자가 페이지 진입 즉시 모든 자산군의 최고 금리 Top 3를 파악하고 원하는 상세 페이지로 이동한다.

**포함 기능 목록**:
- 상단 히어로 배너: 오늘 날짜 + "지금 가장 높은 금리는 N.NN%" 문구 (자동 계산, 전체 자산군 중 최고값)
- 자산군별 Top 3 금리 카드 6개 (CMA / 파킹통장 / 단기채권ETF / 예금 / MMF / 발행어음): 각 카드에 상품명·금리·전일 대비 변동폭(▲▼)·"자세히 보기" 링크 포함
- 전체 통합 랭킹 테이블: 자산군 구분 없이 금리 내림차순 정렬, 상품명·금리·자산유형·원금보호여부·가입채널 컬럼
- 금리 트렌드 미니 차트: 최근 30일 평균 금리 추이 선 그래프 (Chart.js, 데이터는 JSON 파일 참조)
- 데이터 최종 업데이트 타임스탬프 (예: "2026-04-18 오전 9:03 기준")
- 자산군 개념 구분 설명 박스: "확정금리형(원금보장)" vs "시장금리형(원금보장추구)" 2컬럼 텍스트

**데이터 소스**:
- CMA: KOFIA 전자공시 (금융투자협회 금리공시, `http://dis.kofia.or.kr`)
- 파킹통장: 각 은행 API 또는 `pandas.read_html` 스크래핑 (우리·카카오·케이뱅크 등)
- ETF: FinanceDataReader 또는 PyKrX (종목코드: KODEX KOFR=449170, TIGER CD금리=453850)
- 예금: 금융감독원 금융상품통합비교공시 API (`https://finlife.fss.or.kr`)
- MMF: KOFIA 전자공시
- 발행어음: 각 증권사 공시 페이지 (미래에셋·한국투자·KB·NH 4개사)

**SEO 키워드 타겟**:
- 메인: `파킹통장 금리 비교`, `CMA 금리 비교`, `오늘 최고 금리`
- 서브: `단기 고금리 통장`, `안전 자산 금리 랭킹`, `2026 파킹통장 추천`

**Adsense 슬롯 위치**:
- 슬롯 A: 히어로 배너 하단 (728×90 리더보드, 데스크톱) / (320×100 모바일)
- 슬롯 B: 통합 랭킹 테이블과 Top 3 카드 사이 (336×280 직사각형)
- 슬롯 C: 페이지 최하단 푸터 위 (728×90)

**출시 단계**: MVP

### 2-2. `/cma/` — CMA 금리 비교

**목적**: 방문자가 국내 주요 증권사 CMA 상품(발행어음형·RP형)의 금리를 한 테이블에서 비교하여 가입 여부를 즉시 판단한다.

**포함 기능 목록**:
- 필터 탭: "전체 / 발행어음형 / RP형" 3개 탭 (JavaScript 클라이언트 필터, 서버 요청 없음)
- 금리 랭킹 테이블 컬럼: 순위·증권사명·상품유형·금리(%)·전일대비·예치기간·최소금액·가입채널(앱/지점)·최종업데이트일
- 금리 변동 이력 차트: 상위 5개 증권사 30일 금리 추이 선 그래프 (Chart.js)
- CMA 유형별 개념 설명 아코디언: 발행어음형 / RP형 / MMW형 각 2~3문장 + 장단점
- 관련 페이지 링크 박스: "파킹통장과 CMA 비교하기 →", "MMF 금리 보기 →"
- 데이터 최종 업데이트 타임스탬프

**데이터 소스**:
- KOFIA 전자공시 (`dis.kofia.or.kr/websquare/websquare.html`, 금리공시 테이블 `pandas.read_html`)
- 대상 증권사: 미래에셋·삼성·한국투자·NH·KB·신한·키움·대신 (8개사)

**SEO 키워드 타겟**:
- 메인: `CMA 금리 비교`, `증권사 CMA 금리`, `발행어음 CMA 금리`
- 서브: `CMA 발행어음형 추천`, `RP형 CMA 금리 순위`, `2026 CMA 금리`

**Adsense 슬롯 위치**:
- 슬롯 A: 필터 탭 바로 아래, 테이블 위 (728×90)
- 슬롯 B: 테이블과 차트 사이 (336×280)

**출시 단계**: MVP

### 2-4. `/etf/` — 단기채권 ETF 시세

**목적**: 방문자가 KOFR·CD금리 추종 ETF의 당일 시세·수익률·배당금을 확인하고 파킹형 투자 판단에 활용한다.

**포함 기능 목록**:
- ETF 카드 목록: 종목별 카드 (종목명·티커·현재가·전일대비·등락률·52주 수익률·운용보수·설정원본·운용사)
  - 수록 종목: KODEX KOFR금리액티브(합성) 449170, TIGER CD금리투자KIS(합성) 453850, KODEX CD금리액티브(합성) 461980, KBSTAR 머니마켓액티브 494030 (4종 MVP, 추가 종목 v1)
- 30일 기준가격 추이 선 그래프 (Chart.js, 각 ETF 별 개별 차트)
- ETF vs 파킹통장 단순 수익률 비교 표 (동일 기간, 100만원 기준)
- "ETF 투자 시 주의사항" 정적 텍스트 박스 (시장가 매수 슬리피지, 증권거래세 면제 안내 등)
- 데이터 최종 업데이트 타임스탬프

**데이터 소스**:
- FinanceDataReader: `FinanceDataReader.DataReader(ticker, start, end)`
- PyKrX: `stock.get_market_ohlcv_by_date` (백업 소스)
- 운용보수·설정원본: KRX 정보데이터시스템 ETF 종목 정보 API (`data.krx.co.kr`)

**SEO 키워드 타겟**:
- 메인: `KOFR ETF`, `CD금리 ETF`, `파킹형 ETF 수익률`
- 서브: `KODEX KOFR`, `TIGER CD금리`, `단기채권 ETF 추천`

**Adsense 슬롯 위치**:
- 슬롯 A: 카드 목록 상단 (728×90)
- 슬롯 B: 차트 하단 (336×280)

**출시 단계**: MVP

## 6. 빌드 파이프라인 (site-builder 에이전트 요구사항)

1. **입력**: `data/*.json` (data-collector 출력)
2. **템플릿 엔진**: Jinja2 (`templates/` 디렉터리)
3. **출력**: `dist/` 디렉터리에 정적 HTML 파일 (`index.html`, `cma/index.html` 등)
4. **차트 데이터**: `dist/assets/data/` 에 Chart.js용 JSON 복사
5. **CSS**: 단일 `dist/assets/style.css` (모바일 우선, Flexbox/Grid, 외부 프레임워크 없음)
6. **JavaScript**: 단일 `dist/assets/main.js` (필터 탭, 계산기, 툴팁, 햄버거 메뉴)
7. **sitemap.xml / robots.txt**: 빌드 시 자동 생성
8. **GitHub Actions 워크플로**: `.github/workflows/build.yml` — 매일 09:10 KST (`00:10 UTC`) 트리거, data-collector → site-builder → GitHub Pages 배포 순서
